# Car Rental Management System

A console-based Car Rental Management System built in C++ as part of the **Object-Oriented Programming** course. The system streamlines the entire car rental workflow — from booking and returns to invoice generation — using core OOP principles for clean, maintainable design.

## Features

- **Secure Login System** — Restricts access to authorized users only, ensuring data privacy and protecting sensitive records.
- **Role-Based Menus** — Separate, tailored menus for Admins and Users, each offering relevant functionalities for smooth, intuitive navigation.
- **Enhanced Console UI** — A simple yet visually appealing interface, using colored text output to improve readability and overall user experience.
- **Template-Based Design** — Templates are used throughout the codebase to ensure consistency in both functionality and structure.
- **Polymorphism** — Leveraged extensively to make the system more flexible, extensible, and easier to maintain.
- **Robust Error Handling** — Handles invalid inputs and unexpected scenarios gracefully, preventing crashes and ensuring reliable operation.
- **Persistent File Handling** — All data is stored using file handling, so information is retained even after the program closes.
- **Car Inventory Management** — Allows real-time tracking and management of available car quantities.
- **Booking System** — Enables users to make car reservations through a simple, guided process.
- **Return System** — Streamlines the car return process, ensuring accurate and timely transaction completion.
- **Automated Invoice Generation** — Generates invoices automatically upon booking completion, simplifying the payment process.
